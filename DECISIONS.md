# Decisions

| Decision            | Value                                                                                                                                                                                                         |
| ------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Default HTTP port   | 8081                                                                                                                                                                                                          |
| SMTP listener port  | 2525, disabled by default                                                                                                                                                                                     |
| DB default          | `file:./data/agent-mailbox.db`                                                                                                                                                                                |
| Config path         | `./agent-mailbox.config.yaml`                                                                                                                                                                                 |
| Policy path         | `./policy.yaml`                                                                                                                                                                                               |
| Blob storage        | `./data/blobs/`                                                                                                                                                                                               |
| Auth                | API key plus operator token                                                                                                                                                                                   |
| Default policy      | allow by default, external domains require approval                                                                                                                                                           |
| Pagination          | limit 50, max 200, cursor id                                                                                                                                                                                  |
| Webhook retries     | 5 exponential attempts, dead-letter to audit                                                                                                                                                                  |
| Send retry          | 3 attempts                                                                                                                                                                                                    |
| Provider default    | smtp                                                                                                                                                                                                          |
| Telemetry           | none                                                                                                                                                                                                          |
| License             | MIT                                                                                                                                                                                                           |
| Node min            | 22 LTS                                                                                                                                                                                                        |
| First run           | auto-create config and db, print keys once                                                                                                                                                                    |
| Implementation note | The local launch build uses a JSON-backed SQLite-compatible state file so the repo installs without native database compilation; the committed DDL and migration file preserve the Section 7 schema contract. |
